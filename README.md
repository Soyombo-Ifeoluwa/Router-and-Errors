# Router-and-Errors

This is project is a compilation of React and its library.Basically, it is all about:

- Nested Routes.
  -404 pages.
  -Error Boundary.
  -Loading states.
  -Assesibility.
  -Disabled states.
  -Pagination using JSX.

The React Library used in this Project were React-Router-dom and React-Error-Boundary.
